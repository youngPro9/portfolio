The Obstacle Arm

C, Python, Node.js

The Obstacle Arm is the Rover with the robotic arm and IR sensor attached to its top and front respectively. The Rover follows the path retrieved from server. While it's in operation, the IR sensor sends "detected" signal to the Rover if anything is detected in the front within 5 cm distance. Once the Rover receives the "detected" signal, Rover stops moving and communicates with server so that server sends "move obstacle" signal to the robotic arm. The server, then, waits for the "complete" message from the arm. After "complete" message's retrieved, server sends the remaining operation commands to the Rover.

The Flower Detection

Python, Java

The Flower Detection is an Adroid app and is simply a super lightweight botanical dictionary. The app lets a user to take a picture of an interesting flower and retrieve the name of the flower along with a brief description of it within seconds. The server is written in C and hosted on Raspberry Pi 3. The server receives the image of the flower from the user via REST API and anlayze it by using Google Vision API. The name of the flower is taken out of the response from Cloud Vision and is sent to the Wolfram Alpha to get the description of the flower. As soon as the description is received, it's posted to the user's URL and the app displays the name and information of the flower.
